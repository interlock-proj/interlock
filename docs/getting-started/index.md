# Getting Started

Welcome to Interlock! This guide will help you get up and running quickly.

## Prerequisites

- Python 3.10 or higher
- Basic familiarity with Python and async programming

## Installation

=== "pip"

    ```bash
    pip install interlock
    ```

=== "uv"

    ```bash
    uv add interlock
    ```

=== "poetry"

    ```bash
    poetry add interlock
    ```

## Verify Installation

```python
import interlock
print(interlock.__version__)
```
